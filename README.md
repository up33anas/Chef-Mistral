# [!Chef-Mistral]{https://chef-mistral-snowy.vercel.app/}

### A web app that uses AI to suggest recipes based on the ingredients that the user enters.

## Features
- Add at least 4 ingredients to get a recipe from the AI
- Shows a message when user adds duplicate ingredient
- Gives instructions and time too cook the recommended recipe

## Technologies used

- HTML
- CSS
- JavaScript
- React
- API from Hugging Face
